Principais Formatações

Negrito / Destaque

Nesta frase temos uma palavra em negrito usando a tag <b>, (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong>, (semântica).

Itálico / Ênfase

Nesta frase temos um termo em ênfase em itálico usando a tag <i>, (não semântica)

Nesta frase, temos um termo em ênfase usando a tab <em> (com semântica).

Texto Marcado

Podemos criar um texto marcado usando a tag <mark>.

E no outro parágrafo temos um texto marcado no final.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar qu ele deve ser lido, porém não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi inserido depois.
